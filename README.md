## Proyectos destacados

### [Vesta App](https://github.com/ariandasilvaperez/vesta-app) · Finanzas personales (Android)

Aplicación móvil nativa para gestión financiera y control de gastos diarios.

- Panel interactivo con gráficos de gastos y seguimiento del presupuesto mensual
- Funcionamiento **100% offline** con Room (SQLite)
- Arquitectura **MVVM**, Jetpack Compose y Material 3
- Internacionalización completa (inglés / español) y tests unitarios (JUnit)

`Kotlin` · `Jetpack Compose` · `Room` · `MVVM` · `Gradle`

---

### [SpendSense API](https://github.com/ariandasilvaperez/basic-invoices-magement-api) · Backend financiero

API REST central para gestión de facturas, productos y usuarios.

- Arquitectura **Spring Boot** con capas Controller → Service → Repository
- Autenticación **JWT** y control de acceso con Spring Security
- Lectura automática de facturas con **OCR (Mindee)**
- Persistencia relacional con **MySQL** e Hibernate/JPA

`Java 17` · `Spring Boot` · `Spring Security` · `JWT` · `MySQL` · `Maven`
